Class Kawal

Fail2 dalam folder kawal berfungsi 4 cara.

  1. Memilih fail apa yang patut dipaparkan.
  2. Membentuk pembolehubah sendiri
  3. Membentuk tatasusunan sendiri.
  4. Bagi kerja membentuk pembolehubah dan tatasusunan kepada class Tanya

dalam ini, kita boleh gunakan $this->papar dan $this->tanya bergantung kepada keperluan sistem.

class Tanya

Memandangkan Kawal banyak kerja, maka tugas pembentukan tatasusunan dan pembolehubah diserah kepada Tanya

di sini Tanya akan dibantu oleh $this->db dan $this->sql serta $this->insertSql

$this->sql serta $this->insertSql ditugaskan untuk membentuk arahan sql sahaja

tugas $this->db adalah melaksanaan arahan sql dan pulangkan nilai kepada Tanya.

Tanya akan kumpulkan semua nilai dan serah kepada Kawal.

class Papar

dalam class Papar ada fungsi pilihTemplate()

dalam fungsi ini ada beberapa jenis template iaitu

  1. miminium
  2. modern_admin_panel
  3. startbootstrap-sb-admin-2-1.0.8
  4. FlatAdmin-V.2
  5. AdminLTE-2.3.0
  6. A0

A0 adalah template default berasaskan bootstarp twitter 4.1.1

fungsi paparTemplate adalah untuk debug sama ada lokasi include fail adalah fail yang betul

fungsi bacaTemplate adalah untuk memilih fail2 yang berkaitan dengan template

jika $paparFail == false, maka kita akan campak ke \Aplikasi\Kitab\Peta::failPaparTidakWujud()

jika $noInclude == true atau $noInclude == 1, kita hanya include $paparFail sahaja

selain itu baru kita include fail2 berkaitan dengan $paparFail iaitu

  1. diatas.php
  2. menu_atas.php
  3. menu-tengah-atas.php
  4. menu-tengah-bawah.php
  5. dibawah.php

$paparFail diletakkan antara menu-tengah-atas dan menu-tengah-bawah

Konsep asas mvc melayu.

Cara membaca parameter url GET
http://localhost/apps/{kawal}/{kaedah}/(param)/(param)/(param)/(param)/(param)/(param)
url[0] = Kawal -> senarai class dalam folder kawal
url[1] = Kaedah -> senarai fungsi2 dalam class Kawal
url[2] = Param2
url[3] = Param3
url[4] = Param4
url[5] = Param5
url[6] = Param6
url[7] = Param7

Dalam mvc-melayu, kita hanya setkan 6 sahaja pembolehubah dalam setiap fail dalam folder kawal.
Jadi fungsi class Peta adalah mengesan adakah wujud fail atau tidak.
Jika wujud, include fail tersebut guna fungsi spl_autoload_register()
Jika tidak wujud, campak dalam fail Kawal/Sesat.php

Jika fail dalam folder kawal wujud, kita akan setkan 2 nilai secara default.
Maka fail Index dan fungsi index akan diwujudkan secara automatik.
lepas itu kita kan cari fungsi jemaahTaskil() dalam class Kawal.

Tugas class Kawal adalah untuk wujudkan $this->papar dan $this->tanya.
fungsi jemaahTaskil() adalah untuk semak sama ada adalah wujud fail $nama dalam folder tanya.
jika wujud, baru include fail tersebut guna fungsi spl_autoload_register()
